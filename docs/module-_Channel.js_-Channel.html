losePacket(packet) → {boolean}

Loses a packet, preventing it from arriving at its receiver, and firing onPacketLost() with the packet as parameter. If the packet is not currently traveling in this channel, then it does nothing. Returns true if the packet was traveling and was stopped, and false if the packet was not traveling.

onSend(packet, losingPacket, damagingPacket)

Override this method to intercept a packet sent through the channel, to know whether it is going to be lost or not, and if it is going to be damaged. The channel is actually only going to damage the packet if it is not losing it, despite damagingPacket being true.

Parameters:
